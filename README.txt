Adds simple yet attractive responsive social sharing buttons of Facebook, Twitter, Linkedin, Pinterest, WhatsApp and Google+ to wordpress posts, pages or media.

== Description ==

- Social sharing support for Facebook, Google+, Linkedin, Twitter, WhatsApp and Pinterest.
- Auto display to Post, Page, Media and Custom Post type.
- 100% responsive.
- Reorder Social icon order easily.
- Provide Shortcode, can integrated with any theme easily.
You can also use following shortcode

`[toptal-social-share]`

== Installation ==

This section describes how to install the plugin and get it working.
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *Social Share* and click "Install now"
2. Alternatively, download the plugin and upload the contents of `social-sharing.zip` to your plugins directory, which usually is `/wp-content/plugins/`. Activate the plugin.
